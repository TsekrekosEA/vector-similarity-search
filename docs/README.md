# Documentation

This directory contains academic papers and detailed analysis documents for the Vector Similarity Search project.

## Contents

### Academic Papers

**Result_Analysis_Greek.pdf** (Greek)
- Full analysis and benchmarking results from the graduate project
- Comprehensive evaluation of all implemented algorithms
- Performance comparisons on MNIST and SIFT datasets
- Original academic work submitted for CS degree at EKPA (National and Kapodistrian University of Athens)
- For English summary, see [RESULTS.md](../RESULTS.md) in the root directory

**ANN_Part_3.pdf**
- Documentation for Part 3 of the project (Protein Similarity Search)
- Describes the application of ANN algorithms to protein embeddings
- Methodology for comparing ANN methods with BLAST baseline

## For Researchers

If you're interested in reproducing the results or understanding the methodology in detail:
1. Read [RESULTS.md](../RESULTS.md) for an English summary of findings
2. Refer to the Greek PDF for complete academic analysis
3. See [REPRODUCIBILITY.md](REPRODUCIBILITY.md) (when available) for reproduction steps
4. Check algorithm-specific READMEs in `algorithms/` for implementation details
